Zoo Manager - удобная форма с функциями CRUD для животных и зоопарков.


Реализованые функции:

private void ShowZoos() - Показывает все зоопарки

private void ShowAssociatedAnimals() - Показывает всех животных в данном зоопарке

private void listZoos_SelectionChanged(object sender, System.Windows.Controls.SelectionChangedEventArgs e) - Показывает животных в этом зоопарке

private void ShowAllAnimals() - Показывает всех животных из бд

private void DeleteZoo_Click(object sender, RoutedEventArgs e) - Удаляет зоопарк

private void AddZoo_Click(object sender, RoutedEventArgs e) - Добавляет зоопарк

private void AddAnimalToZoo_Click(object sender, RoutedEventArgs e) - Добавляет животного в зоопарк

private void AddAnimal_Click(object sender, RoutedEventArgs e) - Добавляет животного в список

private void DeleteAnimal_Click(object sender, RoutedEventArgs e) - Удаляет животного из бд

private void UpdateZoo_Click(object sender, RoutedEventArgs e) - Обновляет название зоопарка

private void UpdateAnimal_Click(object sender, RoutedEventArgs e) - Обновляет название животного в бд

private void RemoveAnimal_Click(object sender, RoutedEventArgs e) - Удаляет животного

Запустить преокт можно клонировав этот проект к себе 
Далее нужно запустить проект нажатием на f5 или на зеленую кнопку
